```
sudo kubeadm join 52.186.18.198:6443 --token d25t07.xn631slhm9gvx8nn \
```

## Make controller-2 as worker
```
kubectl describe node | egrep -i taint

kubectl taint nodes controller-2 node-role.kubernetes.io/master-

kubectl describe node | egrep -i taint
```
